# Calculus Animator — Hugging Face Space (Docker SDK) dependencies.
#
# This file is the Space-flavored dependency list shipped to the Docker
# container that backs the HF Space. It is intentionally minimal: the
# runtime is FastAPI serving the existing ``ui/`` static bundle plus an
# ``/api/*`` JSON shim that mirrors ``api.bridge.CalculusAPI``.
#
# Gradio / matplotlib / huggingface_hub were removed when the Space SDK
# switched from Gradio to Docker; they have no runtime use here.
#
# pygame is intentionally NOT pinned in this branch yet — slide-render
# endpoints will return ``ModuleNotFoundError`` if invoked. The Dockerfile
# still installs the SDL2 runtime libs so a future ``pygame==…`` line drop-in
# enables ``/api/render_learning_slide`` without an OS-deps rebuild.
#
# Install (Space picks this up automatically; for local Space-mode dev):
#   pip install -r requirements.txt
